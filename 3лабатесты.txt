

Тест кейс №1
Задача: Добавить в корзину  черную мужскую обувь на сайте "https://www.ozon.ru/category/zhenskaya-odezhda-7501/"
1. Зайти на сайт https://www.ozon.ru/category/zhenskaya-odezhda-7501/
2. Ввести в строку поиска "мужская черная обувь
3. Нажать клавишу "Enter"
4. Перейти на понравившийся товар
5. Выбрать черный цвет
4. Нажать на кнопку "добавить в корзину"
Ожидаемый результат: Вы добавили товар в корзину

Тест кейс №2
Задача: найти брюкина  по самой дешевой цене
1. Зайти на сайт https://www.ozon.ru/category/zhenskaya-odezhda-7501/
2. Ввести в строку поиска "брюки"
3. Нажать клавишу "Enter"
4. В строке "Сортировка по:" нажать на вкладку "Цене"
Ожидаемый результат: Вы увидели брюки по самой дешевой цене
